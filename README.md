# Dovesmind Synergy Care

A comprehensive mental health support and substance abuse prevention platform for Nigeria.

## Project info

Dovesmind Synergy Care connects individuals with certified psychologists, rehabilitation centers, and evidence-based training programs.

## Development

This project is built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

### Prerequisites

- Node.js & npm installed

### Getting Started

```sh
# Install dependencies
npm i

# Start the development server
npm run dev
```

## Deployment

Deploy to your preferred hosting provider (Vercel, Netlify, etc.).
